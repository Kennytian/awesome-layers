## 快速开始
可以通过控制台或者 Serverless Devs 工具引用该层。

`sqlite`可执行文件zip包的`bin`目录下下，在函数计算中会解压到 `/opt` 目录下，在函数计算运行环境内，`sqlite`的完整路径是`/opt/bin/sqlite`。
