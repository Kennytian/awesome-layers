## 快速开始

可以通过控制台或者 Serverless Devs 工具引用该层。

使用该层时，需要配置相应的环境变量 (通过控制台创建会自动配置)。

```shell
PATH=/var/fc/lang/python3.10/bin:/usr/local/bin:/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/opt/bin
PYTHONPATH=/opt/python
```
