可以通过控制台或者 Serverless Devs 工具引用该层。

### 基于 Python Runtime 
若选择 Python3.9/Python3.10 Runtime，添加该层后可直接引用

### 基于 Custom Runtime
使用该层时，需要 Python3.7 或以上版本，可参考 Python3.x 公共层。
